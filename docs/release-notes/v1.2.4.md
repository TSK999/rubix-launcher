# RUBIX Launcher v1.2.4

## Clipping settings
- New **Clips** tab in Settings.
- Pick the monitor to record (Auto follows the active game).
- Toggle desktop audio and microphone independently, and choose specific devices.
- Adjustable clip length slider (10s – 120s).
- **Video quality**: resolution (Source / 4K / 2K / 1080p / 720p / 480p) and framerate (24 / 30 / 48 / 60 / 120 fps).
- Changes apply live — the recorder restarts and the pinned monitor updates without relaunching.

## Notes
- Resolution / framerate apply on the next recorder start (press *Arm recorder* or relaunch a game).
- Mic device list reuses the same permission as voice chat, so granting once covers both.
